MuleMakerAHK
============

~~~A Mule Maker For Realm of the Mad God Written in AHK~~~ 
To Make A Mule: Run the .ahk file Where it says Email put a random email for example NiqqaMule (it will automatically add a @yopmail.com to the end) For Password pick a password (must have numbers) Click the drop down menu and click the amount of mules you want Click Make Click the ok button when it tells you your mules there should be a file (Mules.txt) which stores all the mules you make (In Muledump format) Enjoy Your Mules!
